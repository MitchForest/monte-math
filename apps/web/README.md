# @monte/web – Lesson Player Prototype

The web app showcases Monte Math lessons, materials, and data visualizations for stakeholders. It consumes the contract-first API and renders scripted flows with programmatic Pixi scenes.

## Responsibilities

- Display navigation prototypes (overview, knowledge graph, golden beads material, lesson flow).
- Fetch data exclusively through `@monte/api-client` and TanStack Query.
- Render interactive materials (PixiJS) and guided lesson stages.
- Use shadcn/ui + Tailwind v4 for consistent design tokens with the studio.

## Stack Overview

| Concern    | Library                                          | Notes                                     |
| ---------- | ------------------------------------------------ | ----------------------------------------- |
| Build      | Vite + TypeScript                                | Fast HMR, compatible with shadcn CLI      |
| Router     | TanStack Router                                  | Fileless routing with data-prefetch hooks |
| Data Fetch | TanStack Query + `@monte/api-client`             | Contract-typed queries/mutations          |
| State      | Local component state + derived React Query data | Zustand reserved for heavier flows        |
| UI         | Tailwind CSS v4, shadcn/ui (New York theme)      | Components in `src/components/ui`         |
| Canvas     | PixiJS (8.x)                                     | Materials under `src/materials`           |

## Integration Points

- Imports no code from `apps/api`—all calls go through `@monte/api-client`.
- Uses schemas from `@monte/shared` purely for TypeScript inference (lesson script, etc.).
- Does not mutate the graph or content; reads skill/lesson data via the API only.

## Running Locally

```bash
pnpm dev:web         # http://localhost:5173
```

The app expects the API at `http://localhost:3001`. Configure another endpoint through `VITE_API_URL`.

## Folder Guide

- `src/views/` – Page components (overview, graph, material, lesson).
- `src/router.tsx` – TanStack Router definitions + root layout.
- `src/materials/` – Pixi golden bead scene & state helpers.
- `src/styles/globals.css` – Tailwind token layer generated by shadcn.
- `src/lib/orpc-client.ts` – Client factory wrapper (`createApiClient`).

When adding new features, maintain the contract-first pipeline: extend schemas in `@monte/shared`, get the API updated, then call the new procedure via `apiClient` + React Query.
